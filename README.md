# ZigZagRumble-PlayCanvas

ZigZag Rumble is a fast-paced, multiplayer obstacle course game, developed on the PlayCanvas platform. Players race, jump, and dodge through chaotic levels full of moving obstacles, aiming to be first on the finish line.
## Features

* **Multiplayer Action:** Compete against other players in real-time.
* **Dynamic Obstacles:** Each level is filled with spinning platforms, moving walls, and tricky jumps.
* **PlayAnywhere:** Fully web-based, no downloads required.

## How to Play

1. Use **WASD** to move your character.
2. Press **spacebar** to jump.
3. Navigate through the obstacles without falling off.
4. Be first on the finish line to win the round!

## Development

* **Engine:** PlayCanvas
* **Languages:** JavaScript, HTML5
* **Assets:** 3D models and textures optimized for browser performance

### Getting Started

To run the game locally:

1. Create account on [Play Canvas](https://playcanvas.com)
2. Click this this [link](https://launch.playcanvas.com/1981389?debug=true&ministats=true) to start the game
3. Enjoy the game!

### Using this project in your own game
1. Clone this repository:
```bash
git clone https://github.com/misiakfilip/ZigZagRumble-PlayCanvas.git
```
3. Download files.
4. Open PlayCanvas website.
5. Navigate to your projects.
6. Click on import project button.
7. Get creative!
